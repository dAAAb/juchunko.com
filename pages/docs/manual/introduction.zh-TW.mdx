# 寶博簡介

本名葛如鈞，人稱寶博士（簡稱寶博）。

現任不分區立法委員，臺灣大學資訊網路與多媒體研究所兼任助理教授。

## 學歷

2012 年取得臺大資訊工程學博士學位，後赴日本慶応大學媒體設計研究所（KMD）任博士後研究員

## 經歷

曾任日本電通集團台灣 AI/Web3 成長合夥人，也自創知名科技類 Podcast 節目《[寶博朋友說](https://podcasts.apple.com/tw/podcast/%E5%AF%B6%E5%8D%9A%E6%9C%8B%E5%8F%8B%E8%AA%AA/id1484923390)》至今不懈。

## 參與

關於我參與多元民主相關事務和平台的故事，可以從2015年1月開始講起。行政院政務委員兼 vTaiwan 發起人蔡玉玲邀請我和 Audrey 為近百名高級政府官員授課，分享數位政府和虛擬世界的可能性。



當時，我還被邀請擔任行政院虛擬世界法規調適計畫的顧問，並參與推動 [vTaiwan](https://vTaiwan.tw)。

  

同年（2015年）11月，我參與了「政問」線上電視節目的策劃和製作。今年該節目的第一位嘉賓是[前台北市長柯文哲](https://www.facebook.com/talkto.tw/videos/644207609015622/?mibextid=rS40aB7S9Ucbxw6v)，以及[現任桃園市長張善政（國民黨）](https://www.facebook.com/talkto.tw/videos/708509565918759/?mibextid=rS40aB7S9Ucbxw6v)和[現任數位發展部部長唐鳳](https://www.facebook.com/talkto.tw/videos/968165243286522/?mibextid=rS40aB7S9Ucbxw6v)也是嘉賓。目標是促進更透明、開放和跨黨派的政策討論。

  

自2018年以來，我創立了“[寶博朋友說](https://podcasts.apple.com/tw/podcast/%E5%AF%B6%E5%8D%9A%E6%9C%8B%E5%8F%8B%E8%AA%AA/id1484923390)”播客（也在[YouTube](https://youtube.com/@dAAAb)上），邀請了近兩百位來自不同背景和專業領域的傑出人士討論對台灣政策和產業發展有重大影響的技術，如區塊鏈和AI。在眾多集數中，我多次介紹了[Glen Weyl的激進市場](https://www.youtube.com/watch?v=updFgcGXKK4)，並在2020年7月錄製了[與唐鳳的對話](https://www.youtube.com/watch?v=FVA1I4AIOck)。我甚至在2024年邀請了以太坊創始人Vitalik Buterin參加我的播客，並有[Vitalik首次全中文播客](https://www.youtube.com/watch?v=7uSgOOgyV4g)，討論台灣的民主+科技。


2023年11月，我被129歲的國民黨推薦，排名該黨不分區立法委員提名名單第三。根據過 2024/1/13 選舉結果（國民黨得政黨票 476萬4293票，得票率 34.5%），我在2024年2月1日正式進入立法院。


展望未來，我很興奮能夠與Glen Weyl, Vitalik和來自多元宇宙的朋友們[一起促進台灣的多元性](../act/plurality)。
